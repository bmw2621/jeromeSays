Jerome Says API

Usage

GET Request:

Make request to http://jeromesays.gq/ followed by a url encoded string.

Response will be a jpeg of Jerome saying your string


Post Request:

Make request to http://jeromesays.gq/ with "text" parameter as a string

Response be a JSON object with text atrribute giving html link to the above GET request. This was developed to support Slackbot.
